PyValentin

Getting Started with PyValentin

Installation

pip install pyvalentin

Basic Usage

Here's a simple example of how to use PyValentin:

from pyvalentin import MatchMaker

matcher = MatchMaker()
matcher.load_data("your_data.csv")
matches = matcher.create_matches()

Configuration

PyValentin can be configured using a JSON file. Here's a basic example:

{
    "matching": {
        "algorithm": "hungarian",
        "weights": {
            "grade": 0.7,
            "preferences": 0.3
        }
    }
}
